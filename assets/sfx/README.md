# UI Sound Assets

Optional custom sound files for a more tactile CD player feel.

Supported file names:
- `click.mp3` or `click.wav`
- `insert.mp3` or `insert.wav`
- `eject.mp3` or `eject.wav`
- `spin.mp3` or `spin.wav`

If these files are missing, the app falls back to synthesized Web Audio tones.
